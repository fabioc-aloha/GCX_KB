# Skills

> Reusable Copilot skill files for any GCX repo

## Contents

Each subfolder contains a `SKILL.md` that can be adopted into any GCX heir repo's `.github/skills/` folder.

## Structure

```text
skills/{skill-name}/
  SKILL.md          # The skill file
  (optional assets)
```

## Naming Convention

`{skill-name}/SKILL.md` -- lowercase, hyphenated folder name

## Adopting a Skill

1. Copy the skill folder to your repo's `.github/skills/`
2. Adapt any repo-specific references if needed
3. Test that Copilot picks it up in context
