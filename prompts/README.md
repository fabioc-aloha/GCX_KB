# Prompts

> Reusable workflow templates for any GCX repo

## Contents

Standalone `.prompt.md` files that can be adopted into any heir repo's `.github/prompts/` folder.

## Requirements

- Must have `description` in YAML frontmatter
- Must be self-contained
- Should include example usage in the description

## Adopting a Prompt

1. Copy the `.prompt.md` file to your repo's `.github/prompts/`
2. Adapt any repo-specific references
3. Use via natural language in Copilot Chat
